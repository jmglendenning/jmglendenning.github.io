4 Differences

Name the differences between D3.js charts, Shiny and ggplot2

D3.js

A JavaScript library used in the browser, focused on low-level control over web-based graphics and DOM elements. Fully custom, interactive visualizations.

Shiny

An R framework for building interactive web applications. It is not a plotting library, but an enviroment that takes output from R and visualizes it with (often) external libraries like ggplot2.

ggplot2

An R plotting library using the Grammar of Graphics. It creates static or semi-interactive charts, focused on fast, high-quality graphics.
